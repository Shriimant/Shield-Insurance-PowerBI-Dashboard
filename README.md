# Shield Insurance Analytics Dashboard

## Project Overview
This project focuses on analyzing insurance business performance using Power BI. The dashboard provides insights into revenue trends, customer segmentation, sales mode performance, and age group behavior analysis.

---

## Business Objectives
- Revenue trend analysis
- Customer distribution analysis
- Sales mode performance tracking
- Age group behavior analysis
- Business insight generation

---

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

---

## Dataset Information
- dim_customer
- dim_date
- dim_policies
- fact_premiums
- fact_settlements

---

## Data Cleaning & Transformation
- Removed null values
- Corrected data types
- Renamed columns
- Created calculated columns
- Standardized data formats

---

## Data Modeling
Implemented a Star Schema model using:
- Fact tables
- Dimension tables
- Relationship-based analysis

---

## DAX Measures
- Total Revenue
- Total Customers
- Revenue % Change
- Daily Revenue
- Revenue Contribution %
